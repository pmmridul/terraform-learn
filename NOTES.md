# Terraform
- Developed by HashiCorp in 2014.
- Written in Go.
- Scripting in HCL - HashiCorp Language.
- Alternatives: Crossplane, Pulumi
- 'API as Code'


# Lifecycle of Terraform
```
terraform init
terraform plan
terraform apply
terraform destroy
```
# State File
- 'terraform.tfstate'
- To remember current state of infrastructure, while 'applying' or 'destroying'.
